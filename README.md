# About The Repository

This repository contains **Data Analyst Dashboard projects** created to analyze datasets and present insights through interactive visualizations.

These dashboards help businesses understand trends, patterns, and key performance indicators (KPIs) using data-driven insights.

These projects demonstrate **data analysis, dashboard design, and business intelligence skills**.

# ▶️ How To Use

1️⃣ Clone the repository

```
git clone https://github.com/RESH-22/DATA-ANALYST-DASHBOARDS.git
```

2️⃣ Navigate to the folder

```
cd DATA-ANALYST-DASHBOARDS
```

3️⃣ Open any dashboard folder

Example:

```
cd "Spotify Analysis Dashboard"
```

4️⃣ Open the Excel dashboard file to view the interactive dashboard.

---

# Download Individual Dashboard

You can download a specific dashboard folder using:

🔗 https://download-directory.github.io/

### Steps

1. Open the project folder in this repository.
2. Copy the folder URL.
3. Paste it in the website above.
4. Click **Download** to download the folder as ZIP.
